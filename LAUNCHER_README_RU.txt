NIGHTGRAM — ЗАПУСК НА WINDOWS
============================

1. START_NIGHTGRAM.bat
   Обычный запуск для разработки.
   При первом запуске автоматически:
   - проверяет Node.js и npm;
   - создаёт .env.local из .env.example;
   - устанавливает зависимости;
   - запускает сайт на http://localhost:3000;
   - открывает браузер.

2. START_NIGHTGRAM_PRODUCTION.bat
   Локальный запуск production-сборки.
   При отсутствии сборки автоматически выполняет npm run build.
   После изменений в коде удалите папку .next или выполните npm run build вручную,
   чтобы пересобрать приложение.

3. BUILD_PC_INSTALLER.bat
   Заготовка под будущую отдельную Windows-версию NightGram.
   Сейчас проект является веб-приложением, поэтому настоящий EXE-инсталлятор ещё
   не включён. Батник начнёт собирать установщик после добавления Electron/Tauri
   и команды dist:win.

ВАЖНО
-----
Для работы сообщений укажите правильные адреса backend в .env.local:

NEXT_PUBLIC_API_URL=https://ВАШ-BACKEND/api
NEXT_PUBLIC_SOCKET_URL=https://ВАШ-BACKEND

Для первого запуска требуется интернет, чтобы npm скачал зависимости.
