# NightGram 2 — внесённые изменения

- Публичный лендинг отображается сразу и больше не ждёт `/auth/me`.
- Кэшированная пользовательская сессия восстанавливается мгновенно, серверная проверка выполняется в фоне.
- Socket.IO, уведомления и GlobalRuntime монтируются только в авторизованной части.
- Удалена длинная цепочка proxy → wake Railway → fallback → повторный запрос.
- Сетевой тайм-аут сокращён до 9 секунд, refresh-токен повторяется один раз.
- Новый desktop shell: постоянная боковая панель, профиль, настройки и уведомления.
- Новый mobile shell: компактный header, плавающая нижняя навигация и safe-area.
- Убраны полноэкранные анимации при каждом переходе между страницами.

## Переменные окружения

Для прямого подключения укажите:

```env
NEXT_PUBLIC_API_URL=https://YOUR-BACKEND.example.com/api
NEXT_PUBLIC_SOCKET_URL=https://YOUR-BACKEND.example.com
```

Backend должен разрешать origin вашего Vercel-домена через CORS.

## Проверка

```bash
npm ci
npm run type-check
npm run build
```

В среде подготовки архива установка npm-зависимостей не завершилась из-за сетевого тайм-аута, поэтому production build необходимо запустить локально или в Vercel CI.

## NightGram 2.21.0

- Email-коды после правильного пароля.
- Одноразовые резервные коды.
- Включение, отключение и перевыпуск кодов в настройках безопасности.
- Хэширование кодов и challenge-токенов.
- Ограничение попыток и времени действия.

## NightGram 2.21.1

- Email-коды 2FA заменены на стандартные TOTP-коды из приложения-аутентификатора.
- Добавлена локальная генерация QR-кода без сторонних сервисов.
- Добавлены рекомендации 2FAS, Aegis, Microsoft Authenticator и Google Authenticator.
- TOTP-секреты шифруются AES-256-GCM.
- Повторное использование одного временного кода блокируется.
- Сохранены одноразовые резервные коды и управление активными сессиями.

## NightGram 2.22.0

- Безопасное восстановление TOTP 2FA через доверенное устройство.
- Устройство становится доверенным через 24 часа после входа.
- После запроса восстановления действует дополнительная задержка 24 часа.
- Запрос отключает остальные устройства и фиксируется в журнале безопасности.
- Добавлен журнал входов, смен пароля, 2FA, отзывов сессий и удаления аккаунта.
- Добавлено предупреждение о заканчивающихся резервных кодах.
- Добавлен экспорт личных данных в JSON без пароля и секретов 2FA.

## NightGram 2.23.0

- Добавлены отдельные уведомления для личных чатов, групп, каналов и упоминаний.
- Добавлены тихие часы с расписанием через полночь и исключением для @упоминаний.
- Добавлено скрытие текста сообщений в системных уведомлениях.
- Добавлены локальный звук и настройка громкости.
- Веб-версия умеет запрашивать разрешение системных уведомлений браузера.
- Нажатие на сообщение или упоминание открывает точный чат и сообщение.
- Настройки уведомлений синхронизируются через JSONB-поле профиля без новой миграции.
