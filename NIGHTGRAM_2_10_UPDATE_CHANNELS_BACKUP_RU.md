# NightGram 2.10.0 — каналы обновлений и резервные настройки

## Каналы обновлений

В Windows-настройках можно выбрать:

- **Stable** — только обычные публичные GitHub Releases;
- **Beta** — включает предварительные GitHub prerelease-сборки.

Смена канала очищает сведения о предыдущей проверке и запускает новую проверку. Скачивание и установка по-прежнему требуют корректной SHA-256 суммы.

Для beta-релиза используйте версию и тег с суффиксом, например `2.11.1-beta.1` и `v2.11.1-beta.1`. GitHub Actions автоматически отметит такой Release как prerelease.

## Резервная копия настроек

Раздел «Настройки → NightGram для Windows» умеет экспортировать и восстанавливать JSON-файл настроек.

В резервную копию входят:

- оформление NightGram;
- темы отдельных чатов;
- выбранные эффекты магазина;
- аудиовыход;
- параметры запуска, трея и уведомлений Windows;
- выбранный канал обновлений.

Не экспортируются:

- access/refresh tokens;
- пароли;
- аккаунты;
- сообщения и вложения;
- локальная очередь отправки;
- кэш ленты и уведомлений.

## Первый запуск после обновления

NightGram хранит последнюю запущенную версию. После перехода на новую версию приложение:

1. создаёт локальную автоматическую копию desktop-настроек;
2. оставляет не более пяти таких копий;
3. показывает уведомление об успешном переходе версии;
4. предлагает открыть настройки.

Файлы настроек записываются атомарно и имеют резервную копию предыдущего файла.
