# NightGram 2.2.0 — восстановление реального времени

## Что изменено

- Socket.IO переподключается без искусственного ограничения числа попыток.
- При возвращении приложения из сна или восстановлении интернета соединение проверяется немедленно.
- После переподключения активный чат повторно присоединяется к своей Socket.IO-комнате.
- Список диалогов повторно сверяется с сервером: обновляются последние сообщения, статусы присутствия и непрочитанные.
- История открытого чата сверяется с сервером, чтобы восстановить события, пропущенные во время сна или потери сети.
- Локальные сообщения со статусом `sending`/`failed` сохраняются при такой синхронизации и не заменяются серверной историей.
- Presence ping отправляется только когда приложение видно и соединение активно.

## Проверки

- `npm run type-check` — успешно.
- `npm run lint` — успешно, без предупреждений.
