# NightGram 2.12.0 — закреплённые сообщения

## Возможности

- Закрепление и открепление сообщений на ПК и телефоне.
- Панель закреплённых сообщений под шапкой чата.
- Несколько закреплений в одном диалоге.
- Мгновенная синхронизация через Socket.IO.
- Быстрый переход к старому сообщению с загрузкой контекста.
- В личных чатах закреплять может любой участник.
- В группах закреплять могут владелец и администраторы.
- При удалении сообщения закрепление снимается автоматически.

## Миграция

В Supabase SQL Editor выполните `supabase/migration_message_pins.sql`, затем разверните обновлённую папку `backend` на Railway.
