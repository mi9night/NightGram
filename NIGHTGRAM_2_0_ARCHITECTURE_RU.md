# NightGram 2.0 — архитектурная оптимизация

## Что изменилось

### Виртуализированный список чатов
- При большом количестве диалогов в DOM находятся только видимые строки и небольшой запас сверху/снизу.
- Короткие списки отображаются обычным способом, чтобы виртуализация не создавала лишнюю работу.
- Высота списка автоматически пересчитывается при изменении размера окна, поиске и переключении папок.

### Оконный рендеринг сообщений
- Переписка отображает последнее активное окно сообщений, а более ранние части истории добавляются при прокрутке вверх.
- При добавлении старых сообщений позиция прокрутки сохраняется без скачка.
- Поиск автоматически расширяет окно до найденного сообщения.
- Входящее сообщение больше не перебрасывает пользователя вниз, если он читает старую часть переписки.

### Более лёгкая отрисовка
- Удалены отдельные JavaScript-анимации Framer Motion с каждой строки сообщения.
- Вместо них используется короткая CSS-анимация с поддержкой `prefers-reduced-motion`.
- Галерея медиа и размер вложений мемоизированы.
- Видео в сообщениях и информации о чате используют `preload="none"`.
- Обработчики выбора и закрепления чата стабилизированы, поэтому memo-строки не перерисовываются без причины.

### Лента
- Подгрузка следующей страницы выполняется через низкоприоритетный React transition.
- Дубликаты постов при пересечении cursor-страниц удаляются.
- Размер страницы увеличен до 12, а предзагрузка начинается раньше.

### Быстрый и автономный старт
- Удалена обязательная загрузка Google Fonts во время production-сборки.
- Используются системные шрифты Windows/macOS/Linux, поэтому приложение быстрее показывает первый экран и собирается без доступа к fonts.googleapis.com.
- Исправлена старая ошибка с повторяющимися JSX-атрибутами в ChatInfo.
- Исправлена типизация отложенной записи локального кэша.
- TypeScript проверяется один раз перед Next.js build; дублирующая проверка внутри Next отключена.

## Проверки

- `npm run type-check` — успешно.
- `npm run lint` — успешно, без предупреждений.
- Webpack production compilation проходила успешно после удаления Google Fonts. Полная генерация всех страниц в изолированной среде выполнялась слишком долго и была остановлена по лимиту среды; это не сопровождалось ошибкой компиляции или TypeScript.
