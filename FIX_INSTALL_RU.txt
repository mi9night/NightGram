ИСПРАВЛЕННЫЙ ЗАПУСК NIGHTGRAM

В этой версии INSTALL_DEPENDENCIES.bat больше не запускает npm install --include=optional.
Он сразу использует npm install, который самостоятельно синхронизирует
package.json и package-lock.json.

Сообщения вида "npm warn deprecated" являются предупреждениями старых
зависимостей и сами по себе не означают ошибку.

Порядок запуска:
1. Распакуйте архив в новую папку.
2. Запустите START_PC_VERSION.bat.
3. Не закрывайте консоль, пока не появится окно NightGram или явная строка [ERROR].

Если установка ранее оборвалась:
- удалите папку node_modules;
- снова запустите START_PC_VERSION.bat.
