# NightGram 4 — Desktop integration

Добавлено:
- нативные уведомления Windows для событий, пришедших когда окно не в фокусе;
- настройки запуска вместе с Windows;
- переключатель поведения кнопки закрытия: трей или полный выход;
- управление системными уведомлениями;
- сохранение desktop-настроек в профиле пользователя Windows;
- кнопки открытия папки загрузок, перезапуска и полного выхода;
- безопасный IPC-мост через Electron preload с contextIsolation;
- отдельный блок «NightGram для Windows» в настройках уведомлений.

Проверка:
- синтаксис desktop/main.cjs и desktop/preload.cjs проверен через node --check;
- новые npm-зависимости не добавлялись;
- полный TypeScript/build нужно выполнить на Windows через BUILD_PC_INSTALLER.bat.
