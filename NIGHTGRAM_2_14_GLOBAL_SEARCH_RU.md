# NightGram 2.14.0 — глобальный поиск

## Возможности

- единая строка поиска по людям, чатам, сообщениям и файлам;
- фильтры «Всё», «Люди», «Чаты», «Сообщения», «Файлы»;
- поиск выполняется только внутри диалогов, участником которых является текущий пользователь;
- быстрый переход к найденному чату;
- переход к конкретному сообщению с автоматической загрузкой контекста старой истории;
- результаты файлов показывают миниатюру, чат и дату;
- последние запросы сохраняются локально и могут быть удалены пользователем;
- выбранный фильтр запоминается между запусками.

## Backend

После релиза нужно развернуть обновлённую папку `backend` на Railway.
SQL-миграция не требуется.

Endpoint:

```text
GET /api/search/global?q=текст&type=all&limit=12
```

Доступные типы: `all`, `people`, `chats`, `messages`, `files`.
