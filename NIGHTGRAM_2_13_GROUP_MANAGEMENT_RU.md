# NightGram 2.13.0 — управление группами

## Возможности

- Изменение названия, описания и аватара группы.
- Добавление участников через поиск по username.
- Назначение и снятие администраторов владельцем.
- Передача прав владельца другому участнику.
- Удаление участников с проверкой ролей.
- Безопасный выход из группы: владелец сначала передаёт права.
- Системные сообщения о действиях администрации.
- Realtime-синхронизация состава и профиля группы через Socket.IO.
- Полноценная мобильная панель информации о чате.

## Supabase

Выполните `supabase/migration_group_management.sql`. Миграция идемпотентна и добавляет описание группы и индекс ролей участников.

## Railway

После миграции разверните обновлённую папку `backend` на Railway.
