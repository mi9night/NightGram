# NightGram 3.3.1 — состояния звонка и центрирование аватара

## Добавлено

- Цветной статус в верхней части окна звонка.
- Фиолетовый: выполняется вызов.
- Жёлтый: идёт подключение.
- Зелёный: соединение установлено.
- Оранжевый: связь прервалась и восстанавливается.
- Красный: нет сети или соединение отключено.
- Серый: звонок завершён.
- Тот же статус отображается в свёрнутом окне звонка.
- В групповых звонках у каждого участника есть отдельный статус WebRTC-соединения.
- В личном аудиозвонке аватар собеседника занимает центральную плитку на всю ширину; лишняя боковая плитка локального аудио скрыта.

## База и backend

Новая SQL-миграция не требуется. Backend-функциональность не менялась, версия обновлена для единого релиза.
