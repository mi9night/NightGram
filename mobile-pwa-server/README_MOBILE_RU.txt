NightGram 3.4.0 — мобильная PWA

1. Запустите START_MOBILE_SERVER.bat (Windows) или START_MOBILE_SERVER.sh (Linux/macOS).
2. Для проверки интерфейса в одной Wi-Fi-сети откройте на телефоне http://IP_КОМПЬЮТЕРА:3000.
3. Камера, микрофон, установка PWA и Web Push требуют доверенный HTTPS-домен.
4. В приложении откройте Настройки → Уведомления и включите push.
5. Android: Chrome → Установить приложение. iPhone/iPad: Safari → Поделиться → На экран Домой.

Активный звонок восстанавливается после быстрой перезагрузки приложения. При слабой сети NightGram автоматически снижает битрейт видео.
