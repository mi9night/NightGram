# NightGram 2.10.3 — выбор ярлыков в установщике

В пошаговый Windows-инсталлер добавлена отдельная страница «Ярлыки».

Пользователь может независимо выбрать:

- создать ярлык NightGram в меню «Пуск»;
- создать ярлык NightGram на рабочем столе.

При первой установке оба пункта включены по умолчанию, но их можно снять.
При ручном обновлении установщик старается отобразить текущее состояние ярлыков.
Тихие автоматические обновления не меняют существующий выбор пользователя.

Реализация использует поддерживаемый electron-builder NSIS include:
`build/installer.nsh`.

После сборки рекомендуется проверить четыре сценария:

1. оба флажка включены;
2. только меню «Пуск»;
3. только рабочий стол;
4. оба флажка выключены.
