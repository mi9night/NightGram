# NightGram 2.11.1 — реакции, функции сообщений и медиа

## Реакции

- Панель действий на ПК открывается по наведению и закрывается с задержкой.
- Переход мыши от сообщения к панели больше не требует резкого движения.
- На телефоне рядом со временем сообщения есть кнопка действий.
- На touch-устройствах реакции открываются в нижней панели.

## Сообщения

- Свои текстовые сообщения можно редактировать.
- Свои сообщения можно удалить у всех участников.
- Изменённые сообщения получают отметку «изменено».
- Удалённые сообщения остаются в истории как безопасная заглушка.

## Медиа

- Изображения перед загрузкой уменьшаются и конвертируются в WebP.
- Для изображений и видео создаются лёгкие WebP-миниатюры.
- Чаты используют миниатюру до открытия полного файла.
- Видео не загружается заранее и использует постер.
- В базе сохраняются размеры и длительность медиа.

## Обязательное обновление базы

В Supabase SQL Editor выполните:

```sql
supabase/migration_message_edit_delete_media.sql
```

После этого разверните обновлённую папку `backend` на Railway.
