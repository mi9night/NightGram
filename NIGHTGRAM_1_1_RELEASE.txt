NightGram 1.1 Desktop
=====================

Главное изменение: приложение работает без .env.local.
Официальный API и Socket.IO уже настроены на Railway.

Для разработчика:
- START_PC_VERSION.bat — тестовый запуск исходников.
- BUILD_PC_INSTALLER.bat — создаёт готовый Setup.exe в папке release.

Для пользователя после сборки:
- запускается NightGram-Setup-1.1.0-x64.exe;
- Node.js, npm, BAT-файлы и ручная настройка не нужны;
- сразу показывается вход в существующий аккаунт NightGram.
