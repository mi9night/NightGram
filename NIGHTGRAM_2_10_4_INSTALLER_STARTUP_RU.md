# NightGram 2.10.4 — параметры первого запуска и автозапуска

В пошаговый Windows-инсталлер добавлены независимые варианты:

- создать ярлык в меню «Пуск»;
- создать ярлык на рабочем столе;
- запускать NightGram вместе с Windows;
- на финальном экране — запустить NightGram сразу после установки.

Автозапуск по умолчанию выключен. Выбор передаётся приложению через одноразовый файл `installer-preferences.json`, применяется через Electron `app.setLoginItemSettings` при первом запуске и затем удаляется. Тихие автообновления не меняют существующую настройку. При удалении приложения возможные старые записи автозапуска очищаются.
