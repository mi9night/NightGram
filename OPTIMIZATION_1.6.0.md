# NightGram 1.6.0 — Lazy UI chunks

- Комментарии к посту загружаются только при первом открытии.
- Полноэкранный просмотрщик медиа вынесен из стартового пакета ленты.
- Панель информации о чате загружается только по запросу пользователя.
- Экран «Избранное» в сообщениях вынесен в отдельный динамический chunk.
- Сохранены кэширование, отложенный runtime и оптимизация медиа предыдущих версий.
