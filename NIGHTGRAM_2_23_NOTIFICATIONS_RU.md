# NightGram 2.23.0 — уведомления и тихие часы

## Что добавлено

- Отдельные правила для личных чатов, групп, каналов и `@username`.
- Тихие часы по локальному времени устройства, включая интервалы через полночь.
- Возможность пропускать упоминания через тихий режим.
- Отдельные настройки лайков, комментариев, подписчиков и обновлений NightGram.
- Локальный сигнал уведомления и настройка громкости.
- Скрытие текста сообщения в системном уведомлении.
- Системные уведомления Windows/Electron и браузера.
- Переход к конкретному сообщению по нажатию на уведомление.
- Синхронизация настроек между устройствами через существующее поле `users.notification_settings`.

## Миграция

Новая SQL-миграция не нужна. Версия использует уже существующее JSONB-поле `notification_settings` в таблице `users`.

## Обновление

1. Разверните папку `backend` на Railway.
2. Соберите приложение через `BUILD_PC_INSTALLER.bat`.
3. В веб-версии пользователь должен разрешить системные уведомления в браузере.
